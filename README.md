# Markdown Typer

A minimal open-source markdown note-taking app inspired by Raycast Notes.

## Tech Stack

- pnpm workspace
- TypeScript
- NestJS
- Next.js
- Tailwind CSS

## Apps

- `apps/api` — NestJS REST API
- `apps/web` — Next.js web client

## Getting Started

### Install

```bash
pnpm install
```

### Run All Apps

```bash
pnpm dev
```

### Run Checks

```bash
pnpm check
```

## Initial URLs

- Web: `http://localhost:3000`
- API health: `http://localhost:3210/api/health`
